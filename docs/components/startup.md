# Startup

Startup commands configure the basic model settings and must appear at the beginning of the SWAN input file.

## PROJECT

Identifies the simulation with a name and run number.

::: rompy_swan.components.startup.PROJECT

## SET

Sets general model parameters like water level, nautical convention, and coordinate system options.

::: rompy_swan.components.startup.SET

## MODE

Specifies whether the simulation is stationary or non-stationary, and 1D or 2D.

::: rompy_swan.components.startup.MODE

## COORDINATES

Defines the coordinate system (Cartesian or spherical).

::: rompy_swan.components.startup.COORDINATES
